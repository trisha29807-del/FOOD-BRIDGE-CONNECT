# Food Bridge App 🍱

## 📌 Description

Food Bridge is an Android application that connects food donors with people in need, helping reduce food waste and improve accessibility.

## 🚀 Features

* User login & authentication
* Food donation posting
* Browse available food
* Simple and intuitive UI

## 🛠 Tech Stack

* Kotlin
* Android Studio
* XML (UI Design)
* SharedPreferences (for login state)

## 💡 Future Improvements

* Firebase integration
* Real-time updates
* Location-based matching
